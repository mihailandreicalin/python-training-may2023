We will be using pip to install 3rd party packages, so please make sure you can run pip install <package> on your machine:
PyCharm -> Create New Project -> Check Project Interpreter: New Virtual environment -> Create
Once the new project in created, go to Terminal tab and execute the following command:
pip install ipython
The installation should end with a success message: Successfully installed <list of packages>.
